# 법적 체크리스트

## 필수 조치 (서비스 출시 전)

### 1. 저작권 보호 포지셔닝
- [ ] 모든 이미지에 원본 출처 링크 필수 표시
- [ ] 썸네일 200x200 이하 유지 (원본 대체 불가 수준)
- [ ] "검색 엔진" 포지셔닝 유지 (이미지 호스팅이 아님을 명시)
- [ ] 서비스 소개에 "원작자를 찾아 연결하는 서비스" 명시

### 2. 옵트아웃 시스템
- [ ] "내 작품을 삭제해달라" 요청 접수 창구 (이메일 또는 폼)
- [ ] 요청 접수 후 48시간 내 처리 완료
- [ ] 삭제 처리 로그 보관 (분쟁 대비)
- [ ] robots.txt에 크롤러 정보 명시 (해당 시)

### 3. 이용약관 및 개인정보처리방침
- [ ] 이용약관 작성
  - 서비스 이용 규칙
  - 업로드 콘텐츠 권리 및 책임
  - 면책 조항
- [ ] 개인정보처리방침 작성
  - 수집 항목: 이메일, 검색 기록, 브라우징 패턴
  - 수집 목적: 서비스 제공, 추천 개선
  - 보관 기간
  - 제3자 제공 (광고 네트워크)
- [ ] 한국 개인정보보호법(PIPA) 준수

### 4. 광고 표시
- [ ] 광고 콘텐츠에 "광고" 라벨 필수
- [ ] 창작자 프로모션도 유료 시 "광고" 표시
- [ ] 표시광고법 준수

### 5. 콘텐츠 모더레이션
- [ ] 업로드 이미지 NSFW 자동 필터링
- [ ] 불법 콘텐츠 신고 기능
- [ ] 신고 접수 후 24시간 내 검토
- [ ] 아동 성착취물 발견 시 즉시 삭제 + 방통위 신고

## 소스별 법적 상태

### 허용
- X API를 통한 이미지 수집 (API 이용약관 범위 내)
- Safebooru API를 통한 이미지 수집
- Danbooru API (rating:general)
- 수동 브라우징 중 이미지 저장 (일반 이용행위)
- 사용자 직접 업로드

### 주의
- Pixiv: 직접 크롤링 금지, 창작자가 자기 작품을 등록하는 것만 허용
- 이미지 재호스팅: 원본 저장은 저작권 침해 가능, 썸네일만 허용
- 일본 저작권법: Pixiv/booru 출처 이미지 취급 시 일본법도 고려

### 금지
- DC인사이드 자동 크롤링 (이용약관 위반 + 정보통신망법 리스크)
- Gelbooru 무필터 수집 (NSFW/불법 콘텐츠 리스크)
- 이미지 원본 해상도 호스팅 (저작권 침해)
